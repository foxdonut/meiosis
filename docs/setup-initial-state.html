Initial State

When calling meiosisSetup, you can specify an app, with the following properties, all of which are optional:

  • initial
  • services
  • nested

We will look at each of these separately, starting with initial.

By default, the initial state is an empty object, {}. To specify a different initial state, use the initial property of app:

const app = {
  initial: ...
};

const cells = meiosisSetup({ app });

Initial state can also be nested using nested components.
